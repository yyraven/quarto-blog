音乐电台｜千禧年的回响

radio
music
nostalgia
中文
Published

January 14, 2026

怀旧时间到！小时候爸爸经常用千千静听连家庭式音响系统放一些00年代流行的歌曲。这些歌不仅影响了我的音乐品味，也成为了孤身在外时某种意义上的安抚毯。

让一切回到我自己都记不清楚的幼儿园时代去——千禧年，没有移动互联网和串流播放，最后一段家人不怎么出差的日子…

New Age

Karunesh - Punjab

经典德国人写的印度音乐，这首非常美丽。

  • 相关推荐：Karunesh - Zen Breakfast（专辑）

Enigma - Return to Innocence

英格玛这首采样了一位高山族老爷爷的吟唱，好像还闹出了版权纠纷事件，但这种充满张力的吟唱配合着舞曲节奏真的很美。它也是1996年亚特兰大奥运会的宣传曲。英格玛的Modern Crusaders也被《JOJO的奇妙冒险3：星辰远征军》用作了片尾曲，荒木也听New Age啊！

  • 相关推荐：Enigma - Modern Crusaders/Sadeness

Yanni - One Man’s Dream

雅尼给河西走廊之梦创作的主题曲实在太美了，萧瑟而苍凉的气质…

  • 相关推荐: Yanni - 河西走廊之梦

王月明 - 青海青

19年去西藏玩的时候在小巴里一直听着这些音乐。

  • 相关推荐：王月明 - 英吉沙/出塞/紫禁城（专辑）

Enya - Only Time

来自爱尔兰的恩雅的音乐具有浓厚的凯尔特风情，Only Time是其中代表。她的Watermark也是那时候茶餐厅里常常响起的钢琴曲。她还为电影《指环王：护戒使者》创作了美丽的片尾曲May It Be。

  • 相关推荐：Enya - May It Be

偶像剧音乐

KOKIA - 白い雪

《海豚湾恋人》的插曲。

  • 相关推荐：KOKIA - i catch a cold

张韶涵 - 遗失的美好

同样是《海豚湾恋人》的插曲，mv里张韶涵坐在车上丢垃圾，羽绒被里的羽毛撒了一地……

范逸臣 - 国境之南

《海角七号》的插曲。

“当阳光再次/离开那/太晴朗的国境之南/你会不会把你曾带走的爱/在道别前用微笑去归还”

  • 相关推荐：范逸臣/田中千绘 - 野玫瑰；骆集益 - 1945

其他

孙燕姿 - 天黑黑

我爸的车上有一碟孙燕姿的CD。在车上等我妈买菜或者办事的时候我就一直切碟切歌玩儿。

  • 相关推荐：孙燕姿 - 完美的一天

王菲 - 天空

一直知道王菲是妈妈最喜欢的歌手，直到有一天她告诉我是我爸把王菲介绍给她的。我妈喜欢听王菲的抒情曲，尤其是《天空》；小学生们最熟悉的应该是《但愿人长久》；长大后我最喜欢她的摇滚专辑（《只爱陌生人》/《王菲97》/《浮躁》）。

  • 相关推荐：王菲所有的歌都值得一听，我爱她。

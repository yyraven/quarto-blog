  • 重い女

感觉我是那种会说“我要把你永恒地留在我的人生里”的重力系角色，对不起我对所有的朋友们其实都是这么想的。并不期待对方索取我全部的注意力或者期待自己能占据对方所有的注意力，但这条线绝对不能以绝交或者疏远为原因断了。如果真的成为了我心中重要的人，即使分开了我也会看着你一辈子并一直暗暗地期待有一个机会能让人生再次交汇。吵架也无所谓，十年后可以和好吧，十年不行二十年后可以和好吧。分开五年又没关系，我又不是三十岁就会死，分开七年也没关系，我又不是三十二岁就会死（这个statement唯一的缺陷就是我想到有人真的可能会在这五年内去世就会崩溃）。

所有曾经令我感到幸福和感激的时光我都记得一清二楚，并且不会因为后来的矛盾和误会改变，因为我极其自满于自己识人的能力，再怎么变，在那一刻也曾经是我认可的人，并且我相信对方未来依然会是一个我敬佩欣赏的人……

和妹谈心久违地想起了中学时代的心态，觉得自己真的也没怎么变过…极其地傲慢，只是比那时候体面多了…吗

不过体面是真的很重要的，我绝对不会满地乱爬以试图展示忠心以达成少女乐队的毁灭性结局。长崎素世、阿玛兰妲、弗洛伦蒂诺在天上失望地看着我.jpg

  • 到底发生了什么

这么说我已经用不同的理由把对方约出来了，得有，1，2，3，4，5，6？7？次了，同样的理由约出来多次的我自己都记不清有多少次了，先是试探自己的取向和承受度然后又是试探对方的态度和性格，并且需要强调的是这至少十次邀约中每个人都一毛钱多的没花全是A掉的。即使我个人非常不理解靠花钱来定义关系的行为背后的逻辑但也不得不说这可能也是造成迷惑的原因之一。原因之二是我觉得没彻底解除工作联系之前乱谈情感和性都是性骚扰，所以非常礼貌周全圆滑地用其他有趣的话题搪塞过去了，实在是不想触碰这条红线的我就这么在对方谈起相亲和爸妈的校园爱情时说对不起我不太适应谈这些，啊对了我们还是骂一骂老板吧！或者你知道吗你身边这个水泥柱子其实是1918年砌的。或者你知道吗第一个解放自己的黑奴就是从这里离开的哦。或者看啊这个作家娶了自己表妹然后死了，我来考考你到底该用什么疫苗治。或者你来猜猜看我脑子里在想哪个词吧（答案是西瓜虫）。这么个巨长无比的实验周期放在老鼠身上老鼠说不定都死了。

副产品就是我对这的风土人情了如指掌，因为不想去猜对方的态度只能靠高强度学习风土人情转移自己的注意力+不经意地流露出丰富的知识储备以备做一个好导游和别人讲解。

分开后我就没招了，只能像老鼠一样不回对方消息，倒也不是想进行服从性测试主要是一看到消息我就心烦，紧张，这种不确定性让我非常焦虑，但该回的时候还是立正了好好地每一句话都回，就这样又拖了半年，我的生活中又发生了很多事遇见了很多人，朋友们的生活也各有进展，又一如往常地每天都在思考，直到我最近在菩提树下悟道…

…生活在一个周围没有社会常识期待的地方对我很好，坏处就是你难以用常识判断对方行为背后的逻辑是什么（所以我完全理解为何对方迟疑而毫无行动，毕竟我的行为在常识看来也很诡异），好处就是我终于可以把反抗常识的精力收一收放在探索我自己的规则和欲望上了。是吧，算是好处。

  • 双生火焰与All-in-one Pack迷思

这其实就是一句话，即你不可能找到一个既是你的灵魂伴侣又是你妻子又是你妈又是你老师又是你火辣的性幻想对象的人，既然我已经发现自己最核心的情感需求能在朋友身上得到满足并且我也爱我的朋友（这是最近听flamme jumelle的时候意识到的这种程度的感情纠缠或者说共鸣我只有在朋友身上体验过，迷恋其实只是在对方身上放了一面镜子倒映出了我自己），那不如对其他方面放松点…… 2025/02/26

复盘过去一年的心路历程只能说如果Alcest没来我市演出我还能一直纠结到死，谢谢neige，以及过去一年半真是发生了好多事啊，以及我真的很执着，脾气太倔了，以及我就这样单方面宣布胜利了，以我的方式在亲友的支持下做到的胜利，忍不住想请葛城美里、明日香和绫波丽给我鼓掌呢，以及绫波丽出来了neige一定很开心，以及如果我没把上个毛象号清掉的话估计有更完整的心路历程。不过具体的事我都是记在心里的所以也不太可惜…以及我真的不能再这样得意下去了，够了，闭嘴吧！

最后的获奖感言就是世间所有的爱本无区别，学会了爱一个人也就学会了爱所有的人，这第一个人可以是你的家人也可以是你的朋友，没必要等待一个虚无缥缈的想象中的对象才能开始爱，也没必要以不同的标准面对不同“分类”的人，因为本身就没什么分类，一切标准都是人定的，自然也可以手动摘下来。谢谢大家。

即使三天后就掰了我也彻底进化了，心胸开阔了，能看到最远的远方。世间的所有常识唯一一条真的需要在意的就是不要违背对方的意愿性骚扰对方就行了。

  • 再见，所有的少女乐队

这一刻内心最深的渴望其实是回到2023年的5月20号的世纪公园parklife，真我本科四年最无忧无虑的一天，这次我要用苏州话唱义务为豪大大鸡排所作的广告歌，我要大声唱The Smiths和Lou Reed，我要拿吉他弹唱Friday I’m in love和Just like heaven。唱Vapour trail，you’re a vapour trail in a deep blue sky。然后在春日的夜晚看城市里的樱花。当时就觉得特别特别特别幸福。现在回看还是特别特别特别幸福，可以更幸福吗，简直像是∞和2∞的比较，没有意义但是可以更幸福。很单纯很开心。三天后演出，五天后毕业，一个月后去北京找朋友玩，从此以后在此之前的人生仿佛就被封存了一样。从此以后就要操心很多小时候根本不会操心的事情。从此以后少女乐队就结束了。
